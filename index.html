Recently completed my CompTIA Network+ certification. I worked through the topics with lots of labs and practice exams building a stronger foundation in networking basics. I am excited to apply this knowlege and hope to pursue more networking projects.

ESP32 Door Buzzer Project

A small build that connects a text trigger to a physical servo press using an ESP32 and a simple cloud relay. Written using Arduino IDE and Cloudflare Workers. When I text a specific number, the ESP32 receives the signal from Cloudflare and activates a servo to press a door buzzer button.

MyClothes

A web project I have been working on to practice full stack development. This project allows users to upload clothing items and get a randomly generated outfit suggestion based on the weather in the locaiton. Got to try out new frameworks, databases and API integration with this project! Hover over the video to see a quick demo.

Decoding App

A mobile app experiment written in SwiftUI to help a friend who types using the symbol keyboard on her Iphone to encode her messages. Work in progress for sure. Hover over the video to see a quick demo.
